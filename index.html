Tecnologias e outros

  • Python & Django | Django REST Framework

    O backend é feito em Django e expõe uma API REST usando o Django REST.

  • PostgreSQL | PostGIS

    Banco relacional usado no projeto. PostGIS adiciona suporte a features de geolocalização.

  • MongoDB

    Banco de dados de documento usado para criação de pipelines que revelam dados úteis para o cliente.

  • Profiling

    Para controle e redução do uso de recursos do servidor são feitas análises para identificação de gargalos e vazamentos de memória.

  • Integração com serviços de terceiros

    Integração com FireBase, Twilio e AWS

  • Outras tecnologias usadas

    Heroku, Bash, GitLab, Docker, ClickUp
